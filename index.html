Pengantar

Selamat datang di dokumentasi API Lann. API ini menyediakan akses terprogram ke data anime melalui endpoint yang sederhana dan konsisten.

Info: API Lann dirancang untuk memudahkan pengembang mengintegrasikan data anime ke dalam aplikasi mereka dengan cepat dan efisien.

Dengan API ini, Anda dapat mengakses informasi tentang anime yang sedang tayang, anime yang sudah tamat, detail anime, daftar episode, link download, dan masih banyak lagi.

Base URL

Semua endpoint API dapat diakses melalui base URL berikut:

Base URL

https://lnn-api.vercel.app/api

Semua endpoint mengembalikan respons dalam format JSON dan mendukung Cross-Origin Resource Sharing (CORS) untuk memungkinkan penggunaan dari aplikasi browser.

Batas Penggunaan

API ini memiliki batas penggunaan untuk memastikan kestabilan layanan dan ketersediaan untuk semua pengguna.

Tipe Pengguna Batas per Jam Batas per Hari
Umum 100 1,000

Jika Anda melebihi batas penggunaan, Anda akan menerima respons dengan status kode 429 Too Many Requests.

Kode Error

API ini menggunakan kode status HTTP standar untuk menunjukkan keberhasilan atau kegagalan permintaan.

200
OK
Permintaan berhasil diproses
400
Bad Request
Permintaan tidak valid, periksa parameter
404
Not Found
Resource tidak ditemukan
429
Too Many Requests
Melebihi batas rate limit
500
Internal Server Error
Kesalahan pada server
503
Service Unavailable
Server sedang tidak tersedia
